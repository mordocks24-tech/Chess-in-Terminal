## Installation

1. Clone the repository:
   `git clone https://github.com/mordocks24-tech/Chess-in-Terminal`
2. Enter the directory:
   `cd Chess in Terminal`
3. Run the installer:
    Double-click `install.bat`


## How to Play

Once installed, simply open any terminal and type:
```bash
play-chess
